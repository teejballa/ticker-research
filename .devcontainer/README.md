# Devcontainer — Historical (decommissioned)

This devcontainer was used to run the Phase 8/9 container-based research engine
(notebooklm-py on Daytona, then on Google Cloud Run). It was decommissioned in
Phase 12 when the reasoning layer was rewritten as a pure-TypeScript pipeline
(Gemini via the AI Gateway + Firecrawl for community sentiment).

The current app runs entirely on Vercel — no separate container is required.
This file is kept for historical context only.
