# Getting my API Key/Token

How to get my personal API key?

![](https://static.intercomassets.com/avatars/9144211/square_128/FO72F654A4547a21_REV_03-01-1762284091.png)

Written by Xpoz
January 1, 2026

An API key is required in order to connect the Xpoz MCP server with agents such as Claude Code, Gemini CLI, N8N and other agentic platform that don't support OAuth.

You can find your API key in your XPOZ account [settings page](https://www.xpoz.ai/settings):

![](https://downloads.intercomcdn.com/i/o/c47a99k7/1910853798/cbd9ec6d55c1af91a8aa7ee0668d/image.png?expires=1778989500&signature=3f8a7c03828724c7b98720c42ec56dad37414f2d2280fc17d1396d413254cd67&req=dSkmFsF7noZWUfMW1HO4zWke53v4u5vdANw36jsZ74emGcGp8GiCmzN32%2FJg%0At%2FWV5yItQrbEHOcS6zo%3D%0A)
