       - Also call `yahooFinance.quoteSummary(symbol, { modules: ['price', 'summaryProfile'] })` for metadata
       - Return JSON:
         ```typescript
         {
           points: ChartDataPoint[],  // [{time: 'YYYY-MM-DD', value: closingPrice}]
           companyName: string,
           currentPrice: number | null,
           percentChange: number | null,
           marketCap: number | null,
           exchange: string | null,
           sector: string | null,
         }
         ```
       - Wrap in try/catch — return 404 with `{ error: 'Ticker not found' }` if yahoo throws
       - Server-side only. Do not expose API keys.

    2. Create `src/components/PriceLineChart.tsx` (use /frontend-design skill patterns):
       - `'use client'` directive
       - Uses `lightweight-charts` v5 — import `createChart, ColorType, LineType` from `lightweight-charts`
       - Uses `useEffect` + `useRef<HTMLDivElement>` pattern (RESEARCH.md Pattern 4)
       - Chart height: 200px, width: 100% of container
       - Line series using `chart.addLineSeries()` with close prices
       - Call `chart.timeScale().fitContent()` after setting data
       - Returns `() => chart.remove()` in useEffect cleanup
       - Props: `data: ChartDataPoint[]`
       - Handle empty data: render nothing (return null) if data is empty

       NOTE: lightweight-charts v5 changed the series API. Use `chart.addSeries(LineSeries)` if `addLineSeries()` is not available — check the installed version in node_modules to verify the correct method name before using it.
