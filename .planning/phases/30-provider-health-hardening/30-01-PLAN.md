    - `expire(key, seconds)` — set expiresAt = Date.now() + seconds*1000; return 1
    - `lpush(key, ...values)` — unshift values; return new length
    - `ltrim(key, start, stop)` — slice list in place; return 'OK'
    - `lrange(key, start, stop)` — return slice (handle negative stop = -1 = end)
