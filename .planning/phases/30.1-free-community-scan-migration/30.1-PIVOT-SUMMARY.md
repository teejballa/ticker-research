# Phase 30.1 Pivot: Xpoz Replaces Reddit OAuth — Summary

**Replaces the Reddit OAuth + token-cache + rate-bucket plumbing from Plans 30.1-02/03/04 with a thin wrapper over `@xpoz/xpoz` that also unlocks Twitter ingestion (was D-13 deferred). Default flag still `firecrawl` — production untouched until operator flips per the updated playbook.**

## What changed and why

### The trigger

Reddit closed self-service API access in 2025 via the **Responsible Builder Policy**. The policy explicitly bars commercial use, ML training, and aggregate sentiment analysis — Cipher's exact profile. Even with valid client credentials minted under the original 30.1-02 path, application review rejects this use case. The original Plan 30.1 direction was a dead end.

### The replacement

**Xpoz Pro** (`https://xpoz.ai`, $16/mo + $0.80/1K credit overage) is a single unified API for Reddit + Twitter + Instagram + TikTok. Cipher consumes Reddit + Twitter. The TypeScript SDK `@xpoz/xpoz` is on npm. `XPOZ_API_KEY` is already provisioned in Vercel Production + Development env — no infra work needed in this run.

### Strictly-better outcomes vs. the original plan

| Original 30.1 plan | Pivot delivers |
|---|---|
| Reddit only (D-13: Twitter deferred) | Reddit + Twitter via one paid tier |
| Reddit access via OAuth (now TOS-blocked) | Reddit access via Xpoz (no TOS risk) |
| 3-day cron cadence | 6-hour cron cadence (4× freshness) |
| 16-sub roster (some noise, e.g. r/Vitards) | 10-sub roster (signal-quality picks) |
| Per-tier QPM bookkeeping in Upstash | SDK handles quotas server-side |
| No author-authenticity gate | Twitter top-3 citations filtered via Xpoz `isInauthentic` |

## Cost projection

At 6h cron cadence × 10 subs + Twitter + top-3 authenticity gating across 20 tickers + ~30 report-gens/day:

- Reddit: 26,400 q/mo × 2 credits = **52,800 credits/mo**
- Twitter post search: 2,400 q/mo × 2 = **4,800 credits/mo**
- Twitter author authenticity: 7,200 q/mo × 2 = **14,400 credits/mo**
- Report-gen: 10,800 q/mo × 2 = **21,600 credits/mo**
- **Total: ~93,600 credits/mo**
- Cost: $16 + (93,600 − 30,000) × $0.80/1K = **~$67/mo**
