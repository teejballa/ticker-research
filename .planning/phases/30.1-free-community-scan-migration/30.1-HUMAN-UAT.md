```sql
SELECT
  COUNT(*) AS total_calls,
  SUM(CASE WHEN status='error' THEN 1 ELSE 0 END)::float / NULLIF(COUNT(*), 0) AS error_rate,
  CASE WHEN COUNT(*) >= 50 THEN 'eligible' ELSE 'insufficient_history' END AS gate
FROM "ProviderCallLog"
WHERE provider_id IN ('reddit', 'reddit-xpoz')
```

## Per-Provider Verdict Table (D-30)

```sql

  provider_id,
  COUNT(*) AS total,
  AVG(duration_ms) AS p_mean,
  SUM(CASE WHEN status='error' THEN 1 ELSE 0 END)::float / NULLIF(COUNT(*), 0) AS error_rate
```
