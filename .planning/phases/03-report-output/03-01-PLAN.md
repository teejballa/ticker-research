    Create src/lib/formatters.ts with 4 exported functions:

    formatTimestamp(isoString: string): string
    - Use Intl.DateTimeFormat with { month: 'long', day: 'numeric', year: 'numeric', hour: 'numeric', minute: '2-digit', hour12: true }
    - Locale 'en-US'
    - Result format: "March 13, 2026 at 2:32 PM"

    formatMarketCap(value: number | null): string
    - Returns "—" for null/undefined
    - >= 1e12 → "$X.XT" (1 decimal)
    - >= 1e9 → "$X.XB"
    - >= 1e6 → "$X.XM"
    - Otherwise → "$value.toLocaleString()"

    formatPercent(value: number | null): string
    - Returns "—" for null
    - value is a decimal fraction (0.02 = 2%) — multiply by 100 before display
    - Show explicit sign: "+2.34%" or "-1.20%"
    - 2 decimal places

    formatPrice(value: number | null): string
    - Returns "—" for null
    - "$value.toFixed(2)"
  </action>
