**Step 2: Build the recovery counter.** In the same data loader, after fetching LearnedPattern rows, group cells by composite key and add a server-side query for `drift_clear` events in the last 14 days:

```typescript
import { prisma } from '@/lib/db';
// inside the page component / data loader:
const fourteenDaysAgo = new Date(Date.now() - 14 * 24 * 60 * 60 * 1000);
const driftClearCounts = await prisma.learningEvent.groupBy({
  by: ['signal_class', 'pattern_key', 'cap_class', 'horizon_days'],
  where: {
    event_type: 'drift_clear',
    occurred_at: { gte: fourteenDaysAgo },
  },
  _count: { _all: true },
});
const driftClearMap = new Map<string, number>();
for (const row of driftClearCounts) {
  driftClearMap.set(`${row.signal_class}|${row.pattern_key}|${row.cap_class}|${row.horizon_days}`, row._count._all);
}
// Then for each cell row passed into the table:
const recoveryCount = driftClearMap.get(`${cell.signal_class}|${cell.pattern_key}|${cell.cap_class}|${cell.horizon_days}`) ?? 0;
const recoveryReady = cell.status === 'EXPLORATORY-WATCH' && recoveryCount >= 14 && cell.effective_sample_size >= 30;
```
