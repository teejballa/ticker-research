- DATABASE_URL: Neon pooled connection string (from Neon dashboard, has `-pooler` in hostname)
- DIRECT_URL: Neon direct connection string (from Neon dashboard, no `-pooler`)
- DEPLOYMENT_MODE: set to `web` in Vercel environment variables
